Custom Web Components

WireBootstrap for Web Components brings two-day data binding, data services, and application event lifecycle hooks to Web Components for building data driven applications.
The following is a sample button Web Component that binds to the dataset sample dataset. The dataset is executed in the page component and results are sent to the my-button component. It displays values in the Product field in the button label. It calls the btnClick event on the page when any button is clicked.
<my-buttons wr-list="products" label="Product" wr-on-btn-click="btnClick"><my-buttons>
