Wire Components

WireBootstrap for Web Components brings two-day data binding, data services, and application event lifecycle hooks to Web Components for building data driven applications.
The following uses the Wire Web Component to render WireBootstrap's Bootstrap Buttons component. The dataset sample dataset is passed directly to the buttons component and executed there before binding the results to the buttons.
<wire-component component="wire.bsButtons" wr-data="dataset" datamap-id="Product" datamap-label="Product"><wire-component>
